# Sputnik
<h2>
A node/express app built on docker.
</h2>

<h2>
Prerequisites
</h2>
<h4>
Install Dinghy:
</h4>
```
https://github.com/codekitchen/dinghy
```

<h3>
Build docker container
</h3>
```
docker-compose build
```
